Fake API helper

Online REST API for developers.

Auth, Mock, File and Error.

Insomnia docs: download

Base URL

https://fake-api-helper.herokuapp.com

Authentication

Login

POST

/auth

Username: "user", "admin"

Password: "fakeapi"

Request body - application/json *required

{
  "username": "user",
  "password": "fakeapi"
}

Response - 200

{
  "user": {
    "username": "user",
    "name": "User",
    "email": "user@user.sys",
    "photoUrl": "/photos/user.jpg",
    "role": "user",
    "token": "token"
  }
}

Custom mock

GET

/mock/custom?path=folder

Use a DONTPAD folder with JSON content

http://dontpad.com/{folder}

Request header

Name                    Value

Authorization           Bearer token

Request query *required

Name          Value

path          {folder}

Response - 200

{
  "auth": true,
  "role": "user",
  "mock": ...
}

File

Upload

POST

/file/upload

Single file (max. size: 300 KiB)

Download link expiration: 1 hour

Request header

Name                Value

Authorization       Bearer token

Request body - multipart/form-data *required

Name                Value

file                File

Response - 200

{
  "auth": true,
  "role": "user",
  "file": {
    "id": "fileId",
    "url": "/file/download/fileId",
    "fileName": "fileName",
    "fileSize": 123456
  }
}
